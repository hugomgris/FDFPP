\doxysubsection{Detailed Description}
Handles user interaction and controls the viewing transformation. 

The \doxylink{classCamera}{Camera} class manages panning, zooming, and rotating the 3D view. It works together with the \doxylink{classProjector}{Projector} and \doxylink{classRenderer}{Renderer} to adjust how the heightmap is displayed.
